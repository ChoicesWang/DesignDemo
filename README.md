Material design
===========

在Google I/O 发布会中，Google推出了新的设计指南 Material design。这个Dome主要展示 Material design的一些动画效果
和交互设计。

在本Demo用，将会使用使用facebook的最新推出的fresco作为图片加载和缓存框架。https://github.com/facebook/fresco
作为对比，加入了Glide图片加载框架，的确流畅很多。
